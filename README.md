# دليل أمانة السر الكشفية - Quarto Starter

هذا مشروع تجريبي لبناء الدليل باستخدام Quarto.

## 1. تثبيت Quarto

نزّل Quarto من الموقع الرسمي:
https://quarto.org/docs/download/

## 2. فتح المشروع

افتح المجلد في VS Code أو أي محرر نصوص.

## 3. معاينة HTML

```bash
quarto preview
```

## 4. إخراج HTML

```bash
quarto render --to html
```

## 5. إخراج Word

```bash
quarto render --to docx
```

## 6. إخراج PDF

```bash
quarto render --to pdf
```

ملاحظة: PDF العربي يحتاج تجربة خطوط وRTL. إن واجهت مشاكل، ابدأ بالـHTML والـDOCX أولًا.
